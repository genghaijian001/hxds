# Review

- `v2` is a corrected production-oriented version of the single-AppID consolidation plan.
- It explicitly fixes the most dangerous omission in `v1`: identity migration across AppIDs.
- It also formalizes the infrastructure prerequisites:
  - public HTTPS API domain
  - ICP filing
  - WeChat Pay notify URL
  - merchant-to-AppID binding
- This document should be treated as the new baseline for any further implementation planning.
